DOCUMENTATION

The Sci-Fi Lamp Collection package contains a collection of many different detailed lamps. Many lamp prefabs are available in different colors. Thereby a total of 44 prefabs are available.
A few lamps are modular so you can use individual parts of this lamps.
The asset supports all three render pipelines (Builtin RP, URP and HDRP). 
These high quality lamps are perfect for desktops, consoles and high end mobile devices.

If you want to add more details you could use my "Screw Heads Pack - Free".
https://assetstore.unity.com/packages/slug/170388

Please note:

When importing the asset to your project you only need to import the desired render pipeline folder (Builtin RP, URP, HDRP) and the General folder. The Demo folder is optional.
If you switch between Builtin RP demo, URP demo and HDRP demo don‘t forget to remove or add the RenderPipelineAsset in Project Settings -> Graphics -> Scriptable Render Pipeline Settings. 
If you want to use the URP or HDRP demo don't forget to install URP or HDRP from the package manager.

You can find the RenderPipelineAsset in URP/URPResources
and
You can find the RenderPipelineAsset in HDRP/HDRPResources


Texture resolution: 1024 x 1024, 2048 x 2048 and 4096 x 4096


Included:

- Prefabs for all lamps ready to use with Builtin RP, URP and HDRP.
- Texture for all render pipelines. 
- HDRP detail map and normal map textures are included in 2K and 4K.
- Materials for Builtin RP, URP and HDRP.
- Demo scene for Builtin RP, URP and HDRP.
- 3D models.
- Custom PBR shader for URP to use detail mask and detail textures in URP.
